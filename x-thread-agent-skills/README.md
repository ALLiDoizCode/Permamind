## ğŸ¨ Visual Assets

### How to Generate Images

1. Open each HTML file in a browser
2. Set browser window to 1200x800 for optimal Twitter dimensions
3. Use browser screenshot tools or:
   - **macOS**: Cmd+Shift+4, then Space to capture window
   - **Windows**: Snipping Tool or Win+Shift+S
   - **Linux**: GNOME Screenshot or similar
4. Save as PNG with appropriate names (e.g., `01-hero.png`)

### Recommended Image Sizes

- Main tweets: 1200x675 (16:9 ratio)
- Detail shots: 900x900 (1:1 ratio for code examples)
- Banner images: 1500x500 (3:1 ratio)
